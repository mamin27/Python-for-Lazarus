This is fork of Python4Delphi for Lazarus.
Used in CudaText cross-platform editor.
Supports OS:

- Windows (x32 x64)
- Linux (x32 x64 ARM AArch64)
- macOS (x32 x64)
- FreeBSD NetBSD OpenBSD DragonFlyBSD (x32 x64)
- Solaris (x64)
- Haiku (x32 x64)
